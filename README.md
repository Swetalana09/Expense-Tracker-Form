# **Expense Tracker Form**
A simple form to record and track your expenses. You can add what you spent, category, amount, date, payment type, and notes.

**Features**<br>
Expense Details<br>
Date & Time<br>
Payment Methods<br>
Optional Info<br>
Extra Info<br>
Preferences<br>
Validation<br>
Responsive<br>

**Technologies**<br>
HTML<br>
CSS<br>
JavaScript<br>

**Form Validation**<br>
Required Fields: title, category, amount, date, payment, save checkbox<br>
Text: min/max length, no starting spaces<br>
Amount: positive numbers with up to 2 decimals<br>
Transaction ID: letters, numbers, hyphens, underscores only<br>
Shows errors right under the input<br>

**How to Run the project locally**<br>
1. Open the project folder in VS code.<br>
2. Install live server extension.<br>
3. Right click on the index.html<br>
4. Select "Open with Live Server."<br>
5. The Project will open in the browser.
